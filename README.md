# AURUM – Luxury Automobile Website 🚗

AURUM is a modern luxury automobile showcase website designed to provide an elegant user experience for exploring premium cars and booking test drives.

This project demonstrates frontend development skills using HTML, CSS, JavaScript, and Firebase.

---

## ✨ Features

- Luxury automobile showcase interface  
- Modern and responsive UI design  
- Smooth scrolling and animations  
- Test drive booking form  
- Firebase Firestore database integration

JavaScript
